HTML is a hypertext markup language. It is used to create websites. It is the standard markup language for website structure. A website is a collection of web pages and each webpage contains some information in the form of hypertext i.e. text, images, videos, links, etc.

Webpage

A Web page can be seen as an infinite scrollable board on which you have to place different items known as elements these items can be any text, image, video, may be any design, etc.

Element

Element as we above discussed is a part of the webpage. It can be seen as boxes or items that are placed in different locations on a webpage. The element in general contains the content and this type of content is known as a tag. The tag specifies the type of the content it encloses. The element is generally enclosed by the opening and closing tag.

<div> Content Here </div>
